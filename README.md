# 🧮 Math Game - Quiz Master

### Description
A simple Math Quiz Game built with **C# Windows Forms**.  
You can select the operator type (Addition, Subtraction, Multiplication, Division), difficulty level (Easy, Medium, Hard), and the number of rounds to play.

---

### How to Run
1. Open the project in **Visual Studio**.  
2. Run the application.  
3. Click on **Settings** in the top menu:  
   - Choose **Operator Type**.  
   - Choose **Difficulty Level**.  
   - Enter **Number of Rounds**.  
4. Press **Start Round** to begin.  
5. Select the correct answer and press **Check**, then move to the next question with **Next**.  
6. You can reset the game using the **Reset** button.  

---

### Features
- Multiple difficulty levels: Easy, Medium, Hard, Mix.  
- Different operator types (+, -, ×, ÷).  
- Win/Fail counter.  
- Progress bar showing completion percentage.  
- Error messages and alerts for invalid settings or finished rounds.  

---

### Requirements
- Visual Studio 2022 or later.  
- .NET Framework 4.7.2.
